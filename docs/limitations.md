# Limitations

AlphaForge is an educational research platform, not a production trading system.

Important limitations:

- yfinance data can contain survivorship, adjustment, and availability issues.
- Synthetic data verifies engineering behavior but is not market evidence.
- Backtests are close-to-close approximations and do not model order book dynamics.
- Slippage and spreads are simplified.
- No broker integration places live orders.
- Model results can overfit even with walk-forward validation.
- Real deployment would require stronger data licensing, monitoring, capital controls, compliance review, and independent validation.
