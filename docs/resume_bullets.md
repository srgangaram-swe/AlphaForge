# Resume Bullets

- Built AlphaForge, an end-to-end quantitative ML research platform for public market data ingestion, feature engineering, multi-horizon return labeling, walk-forward validation, OOS prediction generation, portfolio construction, and realistic backtesting.
- Engineered leakage-safe time-series ML workflows with embargoed train/test splits, causal feature tests, and CI checks that mutate future data to verify feature invariance.
- Implemented configurable alpha models across baselines, regularized linear models, tree ensembles, optional PyTorch sequence models, and prediction ensembles.
- Designed a vectorized backtesting engine with next-bar execution, turnover-based commissions, spread/slippage assumptions, volatility targeting, portfolio caps, and risk/performance analytics.
- Added reproducible CLI workflows, configs, Docker support, tests, CI, a Streamlit dashboard, FastAPI endpoints, and simulated paper-trading replay.
